Добро пожаловать в исходный код программы "Fractals"

 ----------------------------------------------------------------------------------------------

Программа разделена на 3 логических блока:

	1)MainWindow.xaml и MainWindow.xaml.cs - содержит описание дизайна и код собтий.
	2)Fractals - содержит набора классов, разделённых на файлы. В них содержится исходный
	  код всех классов фракталов.
	3)Tools - содержит два вспомогательных класса работы програмы один для построения фракталов,
	  другой для сохранения фракталов.

 ----------------------------------------------------------------------------------------------
 -Особености-

 1) Размер холста зависит от вашего разрешения экрана и равен его разрешению,
    следовательно выходная картика сохранения будет зависить от разрешения вашего экрана.
    Холст протестирован на экранах разрешения HD, FULL HD и QUAD HD.
 2) Приближение происходит в центр холста, дльнейшую корректировку положения следует производить с
	помощью ползунков вокруг видимой картинки.
 3) Сохранение производится в папке, в которой нахордится .exe, по пути ./Saves в формате .png c альфа-каналом.
 4) Архитектура классов фракатлов исполнена в соответствии с ТЗ.
 5) Рекомендуется иметь на машине 8 и больше гигабайт оперативной памяти, а также процессор не старше 2016 года.
 6) Потеряли фрактал из виду? Ещё раз нажмите на кнопку "построить фрактал" или воспользуйтесь ползунком перемещения!
 7) Изначально дополнительные настройки фракталов Дерево и Множество Кантора установлены на стандартные настройки.
 8) Изначально основные настройки установлены на стандартные значения для быстрого построения фрактала.

 -Как работать с программой-

 1) Выберите тип фракатала кнопкой.
 2) Произведите выбор входных данных с помощью ползунков.
 3) Нажмите кнопку "построить фрактал".
 4) Можете вопользоваться ползунками вокруг картинки фрактала, чтобы перемещаться по холсту.
 5) Испорльзуте ползунок масштаб для того чтобы приблизить картинку.
 6) Используйте ползунок рекурсии чтобы изменить глубину, фрактал автоматически перерисуется.
 7) Сохраните фрактал нажатием на кнопку "сохранить".
 8) Выберите другой тип фрактала, не забывайте, что ползунок масштаба остаётся в том же положении,
	следовательно следующий фрактал будет построен с тем же приближением.

 -Ограничения-

 Для каждого фрактала установлены разумные ограничения глубины рекурсии и длин отрезков,
 если вы хотите проэкспереминтировать, пожайлуста измените входные данные метода SetFractalInterface.

 -Сохранение фракталов-

 1) Путь до папки: ./Saves.
 2) Название файла включает тип фрактала и время его создания.
 3) Формат сохранения .png.

 -Реализованный дополнительный функционал-

  Тип 4 - Открыто вы можете встретить MessageBox только при попытке сохранить фрактал без самого фрактала или при ошибке файловой системы компьютера.
  Вероятно вы сможете вастретить ошибку с MemoryOverflow, если на вашей машине критически мало оперативной памяти, это приведет к тому,
  что вам предложат изменить входные данные и очень маловероятно встретить всплывающее окно при других ситуациях, хотя обработка всех исключений предусмотрена. 

  Тип 5 - не имеет смысла при подобном интерфейсе и архитектуре, исполнении Тип 8, хотя соблюдено требование с размерностю окна и расширением интерфейса.

  Тип 6 - Реализован выбор начальных и коненчых цветов градиента,
  градиент реализован для каждого типа фрактала.

  Тип 7 - Предусмотрено плавное увеличений картинки до x5.

  Тип 8 - Предусмотрена возможность прокрутки изображения как в увеличеном так и в обычном режиме.

  Тип 9 - Предусмотрена возможность сохранения фрактала, формат выбран самостоятельно (.png)


Приятного использования!
