Добро пожаловать в исходный код программы "Notepad+"

 ----------------------------------------------------------------------------------------------

Программа включает 4 формы:
 1)FormMain - содержит код основной формы. 
 2)FormSettings - cодержит код формы настроек.
 3)FormFontSettings - содержит форму настроек шрифта. 
 4)FormMiniFontSettings - cодержит уменьшенную форму настроек шрифта.

 ----------------------------------------------------------------------------------------------

|====== Ограничения работы программы 

--- Размерность
1) Файлы размером не более 10 мб.
2) Вкладок в одном окне - 10 штук.

-- Особености работы
1) При закрытии родительской формы, закрываются все формы, это никак не влияет на открытие в будущих сессиях.
2) Приложение не производит автоматическое сохранение безымянных файлов.
3) Использование функциии "Сохранить всё" предлагает сохранить также и изменёные безымянные файлы,
   если это не автоматическое сохранение.
4) Журнал автосохранений автоматически чистится при закрытии приложения и для каждого файла отдельно при его закрытии вручную. или открытии нового файла в той же вкладке.
5) Журнал автосохранений не ведется для безымянных файлов.
6) Журнал автосохранений сохраняет 10 последних версий файла с учётом заданного расширения.
7) При открытии главного меню ненужные элементы интерфейса исчезают.
8) Существует возможность открыть множество вкладок настроек, тогда настройки будут приняты из последнего окна,
   в котором вы нажали кнопку "принять".
9) Для каждого открытого окна ведется своя папка автосохранения.
10) Отмена и поаторения действия работают только на один шаг назад или вперёд.
11) Автоматическое форматирование кода производится по нажатию кнопки "Форматировать код".
12) Если файл был изменен, но не сохранён, перед его названием появляется "*".

--- Шрифт
1) При использовании двух разных шрифтов из разных семей, при выделении одновременно двух типов семей
   невозможно применить стиль текста (жирный, курсив...).
   Форматирование текста доступно только для выделенной области с шрифтом одной семьи.

--- Компиляция файлов
1) Компилировать и форматировать возможно только файлы .cs, сохраните свой Безымянный файл в формате .cs,
   чтобы получить доступ к форматированию и компиляции.
2) Вывод ошибок поддерживает только символы кодировки 866, то есть приспособлен для использования в русской версии Windows.
3) Для компиляции в настройках требуется указать путь до csc.exe, его вы можете найти сами в системе,
   можно использовать даже более ранние версии компилятора,
   если они поддерживают синтаксис используемый в вашем файле с кодом.
   Однако рекомендуется использовать компиляторы в папке: 
   C:\Windows\Microsoft.NET\Framework64
4) Вывод исполняемого скомпилированого файла совершается в основную библиотеку приложения,
   путь до нее будет указан в сообщении при успешной компиляции.
