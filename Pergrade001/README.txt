Добро пожаловать в исходный код игры "Быки и коровы"! 
Кода разделен на логические модули - файлы:
 1)Bulls_and_Cows.cs - содержит скелет игры (основной файл). 
 2)Enviroment.cs - содержит элементы интерфейса программы! Методы там имеют размеры больше рекомендованых из-за большого колличества графической информации, не бейте за это...
 3)Game.cs - содержит класс с методами создания игры. 
 4)Input.cs - содержит класс с методами для проверки корректности ввода пользователя.
 5)Tools.cs - cодержит генератор чисел и инструменты работы с массивами.
