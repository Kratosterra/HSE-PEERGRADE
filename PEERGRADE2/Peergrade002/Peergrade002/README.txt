Добро пожаловать в исходный код Калькулятора матриц!

Кода разделен на логические модули - файлы:
 1)MatrixCalculator.cs - содержит скелет игры (основной файл). 
 2)Enviroment.cs - содержит элементы интерфейса программы! Не бейте за многа многа текста...
 3)SelectAction.cs - содержит класс с методами выбора действия калькулятора. 
 4)Actions.cs - содержит скелет каждого доступного действия. 
 5)Input.cs - содержит класс с методами для проверки корректности ввода пользователя.
 6)WorkInputConstruct.cs - cодержит классы для создания архитектуры последовательности ввода пользователя.
 7)MatrixVerification.cs - cодержит класс, проверяющий сами матрицы.
 8)Tools.cs - cодержит методы решения действий с матрицами.

 Как создать input.txt?

 1)После первого прогона программы, когда ваша VisualStudio создаст в этом проекте папку bin, зайдите в нёё, далее в папку Debug.
 2)Щёлкните правой кнопкой мыши и создайте текстовый файл в этой директории.
 3)Откройте файл, запишите в него число и две матрицы через разделители ";" (Как это сделать вы увидете в Как должен выглядеть input.txt.png)
 4)Просто точно повторите всё как там, размерность матриц, все значения, конечно же, можно менять! Главное придерживаться струкутры!
 5)Нажмите CTRL+S, чтобы сохранить изменения)

 Как используются данные из input.txt?

 Число, конечно же, используется для домножения на первую матрицу!
 Первая матрица является основной, в отношении нёё производится большинство действий!
 Вторая матрица используется для произведения, сложения, вычитания и всего, что требует две матрицы!
