Добро пожаловать в исходный код “Транспорт”!

----------------------------------------------------------------------------------------------

Кода разделен на логические модули - файлы:

Библиотека EKRLib:
 1)Transport.cs - содержит абстрактный класс, репрезентующий Транспорт.
 2)Car.cs - содержит класс-наследник от Transport, репрезентующий автомобиль.
 3)MotorBoat.cs - содержит класс-наследник от Transport, репрезентующий моторную лодку.
 4)TransportException.cs - содержит сериализуемый класс-наследник от-+ Exception, содержпщий в себе 4 конструктора, включая
   конструктор сериализации.
   
Консольное приложение:
 1)Program.cs - содержит скелет программы (основной файл). 
 2)Environment.cs - содержит элементы интерфейса программы.
 3)Testing.cs - содержит методы для тестирования библиотеки. 
 4)TransportCreation.cs - содержит класс с методами для создания двух видов трансопрта.
 5)Tools.cs - содержит методы, которые выполняют вспомогательную роль в работе программы.

-----------------------------------------------------------------------------------------------
Особености:

Программа пререодически генерирует модель транспорта несоответсвующую спецификации, чтобы продемоснтрировать работу
проверки в сеттере класса Transport.
